# coc-sonarlint

sonarlint for coc.nvim

## Install

`:CocInstall coc-sonarlint`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-sonarlint-keymap)`

## Lists

`:CocList demo_list`
